A special audio visualizer to generate videos. 

_Primarily designed for launchpad performance videos._

> [!WARNING]
> This project is under construction.

## Installation

FFmpeg is required, download it at https://www.ffmpeg.org/ and config the path in the app.

Pre-release is coming soon. 

## Get the audio stems

You can split the audio into stems using:

- iZotope RX series(9/10/11/12/more).
- [Ultimate Vocal Remover v5](https://ultimatevocalremover.com/)

or more tools.

## Dev

Clone this repo, and:

```bash
# or npm, yarn, bun, etc.
```
